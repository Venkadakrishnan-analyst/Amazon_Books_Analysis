## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Project Objectives
- Perform exploratory data analysis (EDA)
- Identify trends in bestselling books
- Uncover key insights through simple and easy-to-understand visualizations.
- Practice data analysis using Python

## Dataset
- Amazon Best Selling Books Dataset (2009–2019)


## Insights Extracted
- Top 10 and Bottom 10 authors based on the number of bestselling books.
- Price distribution of books.
- Average book ratings by category (Fiction and Non-Fiction).
